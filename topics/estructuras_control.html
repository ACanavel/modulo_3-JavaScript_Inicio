Introducción a las Estructuras de Control

Las estructuras de control permiten alterar el flujo de ejecución de un programa en función de ciertas condiciones. En JavaScript, disponemos de estructuras condicionales y de bucles para repetir tareas.

Estructuras Condicionales

Sentencia if

La sentencia if ejecuta un bloque de código si una condición es verdadera.


// Ejemplo de uso de if
let edad = 18;
if (edad >= 18) {
    console.log("Eres mayor de edad");
}

Referencias: if...else - MDN Web Docs

Sentencia if...else

Se utiliza para ejecutar un bloque de código si la condición es falsa.


let esEstudiante = false;
if (esEstudiante) {
    console.log("Tienes descuento de estudiante");
} else {
    console.log("No tienes descuento");
}

Sentencia else if

Permite verificar múltiples condiciones.


let nota = 85;
if (nota >= 90) {
    console.log("Calificación: A");
} else if (nota >= 80) {
    console.log("Calificación: B");
} else {
    console.log("Calificación: C");
}

Sentencia switch

La sentencia switch evalúa una expresión y ejecuta el código correspondiente al caso coincidente.


let diaSemana = 3;
switch (diaSemana) {
    case 1:
        console.log("Lunes");
        break;
    case 2:
        console.log("Martes");
        break;
    case 3:
        console.log("Miércoles");
        break;
    default:
        console.log("Otro día");
}

Bucles o Ciclos

Bucle for

Se utiliza para iterar sobre un bloque de código un número conocido de veces.


// Imprimir números del 1 al 5
for (let i = 1; i <= 5; i++) {
    console.log(i);
}

Referencias: for - MDN Web Docs

Bucle while

Ejecuta un bloque de código mientras una condición sea verdadera.


let contador = 1;
while (contador <= 5) {
    console.log(contador);
    contador++;
}

Referencias: while - MDN Web Docs

Bucle do...while

Similar al bucle while, pero la condición se evalúa después de ejecutar el bloque de código al menos una vez.


let numero = 6;
do {
    console.log(numero);
    numero++;
} while (numero <= 5);

En este caso, aunque la condición es falsa desde el inicio, el código dentro del do se ejecuta una vez.

Referencias: do...while - MDN Web Docs

Control de Bucles

Sentencia break

Termina el bucle actual inmediatamente.


for (let i = 1; i <= 10; i++) {
    if (i === 5) {
        break; // Sale del bucle cuando i es 5
    }
    console.log(i);
}

Sentencia continue

Omite la iteración actual y continúa con la siguiente.


for (let i = 1; i <= 5; i++) {
    if (i === 3) {
        continue; // Salta la iteración cuando i es 3
    }
    console.log(i);
}
