Set

Un Set es una colección de valores únicos.Un Set es como una lista (o array), pero con una diferencia importante: no permite que haya valores repetidos.


const conjunto = new Set();
conjunto.add(1);
conjunto.add(5);
conjunto.add(1); // No se añade, ya que 1 ya existe

console.log(conjunto); // Salida: Set {1, 5}

Symbols

Los Symbol son identificadores únicos e inmutables.


const id = Symbol('id');

const usuario = {
    nombre: 'Ana',
    [id]: 12345
};

console.log(usuario[id]); // Salida: 12345

Proxy y Reflect

Los Proxy permiten crear un objeto que intercepte y redefina operaciones básicas.


const objetivo = {};
const handler = {
    get: function(obj, prop) {
        return prop in obj ? obj[prop] : 42;
    }
};

const proxy = new Proxy(objetivo, handler);

console.log(proxy.a); // Salida: 42

El objeto Reflect proporciona métodos estáticos que corresponden a los interceptores de los proxies.
