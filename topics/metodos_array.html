Métodos Clave de Array

map()

Crea un nuevo array con los resultados de llamar a una función por cada elemento del array.


const numeros = [1, 2, 3, 4];
const cuadrados = numeros.map(num => num * num);

console.log(cuadrados); // Salida: [1, 4, 9, 16]

filter()

Crea un nuevo array con todos los elementos que cumplan la condición implementada por la función.


const edades = [18, 22, 15, 30, 17];
const mayoresDeEdad = edades.filter(edad => edad >= 18);

console.log(mayoresDeEdad); // Salida: [18, 22, 30]

reduce()

Aplica una función a un acumulador y a cada valor del array para reducirlo a un único valor.


const precios = [10, 20, 15];
const total = precios.reduce((acumulador, precio) => acumulador + precio, 0);

console.log(`Total: $${total}`); // Salida: Total: $45

forEach()

Ejecuta la función indicada una vez por cada elemento del array.


const frutas = ['Manzana', 'Banana', 'Naranja'];
frutas.forEach(fruta => console.log(fruta));
