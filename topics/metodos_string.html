Métodos Recientes de String

trimStart() y trimEnd()

Elimina los espacios en blanco del inicio o final de una cadena.


const texto = '   Hola Mundo   ';
console.log(texto.trimStart()); // Salida: 'Hola Mundo   '
console.log(texto.trimEnd());   // Salida: '   Hola Mundo'

replaceAll()

Devuelve una nueva cadena con todas las coincidencias de un patrón reemplazadas por una cadena de reemplazo.


const frase = 'El perro es el mejor amigo del perro';
const nuevaFrase = frase.replaceAll('perro', 'gato');

console.log(nuevaFrase); // Salida: 'El gato es el mejor amigo del gato'

Separadores Numéricos

Mejoran la legibilidad de números grandes utilizando el guión bajo _ como separador.


const numeroGrande = 1_000_000_000;
console.log(numeroGrande); // Salida: 1000000000
