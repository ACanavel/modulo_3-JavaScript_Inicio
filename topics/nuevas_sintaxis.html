Fusión Nula (??)

El operador de fusión nula devuelve su operando del lado derecho cuando el operando del lado izquierdo es null o undefined.


let usuario;
let nombre = usuario ?? 'Invitado';

console.log(nombre); // Salida: 'Invitado'

Encadenamiento Opcional (?.)

Permite acceder a propiedades anidadas sin causar un error si una referencia intermedia es null o undefined.


const persona = {
    nombre: 'Carlos',
    direccion: {
        ciudad: 'Madrid'
    }
};

console.log(persona.direccion?.ciudad); // Salida: 'Madrid'
console.log(persona.contacto?.telefono); // Salida: undefined
