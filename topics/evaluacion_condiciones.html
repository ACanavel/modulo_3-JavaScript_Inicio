Operadores Lógicos en JavaScript

Los operadores lógicos se utilizan para combinar condiciones y evaluar expresiones booleanas. Los principales operadores lógicos en JavaScript son:

Operador AND (&&)

Devuelve true si ambas expresiones son verdaderas.


let edad = 25;
let tieneLicencia = true;

if (edad >= 18 && tieneLicencia) {
    console.log("Puede conducir");
} else {
    console.log("No puede conducir");
}

Operador OR (||)

Devuelve true si al menos una de las expresiones es verdadera.


let esAdmin = false;
let esEditor = true;

if (esAdmin || esEditor) {
    console.log("Acceso permitido");
} else {
    console.log("Acceso denegado");
}

Operador NOT (!)

Invierte el valor de una expresión booleana.


let esVisible = false;

if (!esVisible) {
    console.log("El elemento está oculto");
}

Operador Ternario

Es una forma concisa de escribir una sentencia if...else.


let puntaje = 80;
let resultado = (puntaje >= 70) ? "Aprobado" : "Reprobado";

console.log(resultado); // Salida: Aprobado

Referencias: Operador Condicional (Ternario) - MDN Web Docs

Precedencia de Operadores

La precedencia determina el orden en que se evalúan los operadores. Es importante usar paréntesis para asegurar el orden deseado.


let resultado = true || false && false; // && tiene mayor precedencia que ||
console.log(resultado); // Salida: true
